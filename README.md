# 🤖 WordMaster Pro AI

Telegram-бот для изучения иностранных слов с использованием **Google Gemini AI**.

## ✨ Возможности

-  **AI-генерация слов** — Google Gemini подбирает слова под ваш уровень
- 🌍 **8 языков** — английский, испанский, немецкий, корейский, французский, японский, итальянский, китайский
- 📊 **Адаптивная сложность** — 3 уровня (A1-C2)
- 📝 **Интерактивные тесты** — 5 вопросов за раз
-  **Статистика** — отслеживание прогресса
- 🎯 **Персонализация** — AI учитывает ваш уровень
