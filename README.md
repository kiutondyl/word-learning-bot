#  Language Learning Bot

Telegram-бот для изучения иностранных слов с системой тестирования и статистикой.

## ✨ Возможности
- 🌐 3 языка: английский, испанский, немецкий
- 📖 Режим обучения с примерами использования
- 🧠 Интерактивные тесты с вариантами ответов
- 📊 Персональная статистика прогресса
- 💾 Сохранение прогресса в базе данных

## 🚀 Запуск
1. Клонируйте репозиторий
2. Установите зависимости: `pip install -r requirements.txt`
3. Получите токен у @BotFather
4. Создайте `.env` файл: `BOT_TOKEN=ваш_токен`
5. Запустите: `python main.py`

##  Технологии
- Python 3.10+
- pyTelegramBotAPI
- SQLite
- python-dotenv

## 📱 Команды
- `/start` — начать и выбрать язык
- `/learn` — выучить новое слово
- `/quiz` — пройти тест
- `/stats` — статистика
- `/reset` — сбросить прогресс
